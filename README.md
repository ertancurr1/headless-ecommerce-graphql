# Headless E-Commerce GraphQL

A production-style headless e-commerce platform built with modern technologies.

## Tech Stack

- **Frontend:** React + TypeScript + Apollo Client + SCSS
- **Backend:** PHP 8 (OOP, Clean Architecture)
- **API:** GraphQL
